# Site images

Drop your photos here. Files in `public/images/` are served at `/images/...` and referenced by the app automatically.

Replace placeholder filenames with your own JPG or WebP files (keep the same names, or update paths in `src/lib/siteImages.ts`).

## Folder layout

| Folder | File | Used for |
|--------|------|----------|
| `hero/` | `background.jpg` | Hero section backdrop (1920×1080 recommended) |
| `about/` | `background.jpg` | About section subtle backdrop |
| `about/` | `founder.jpg` | Founder portrait (4:5 crop works well) |
| `services/` | `planted-tanks.jpg` | Planted Tanks service card |
| `services/` | `koi-ponds.jpg` | Koi Ponds service card |
| `services/` | `rainforest.jpg` | Rainforest Vivariums service card |
| `services/` | `terrariums.jpg` | Terrariums service card |
| `services/` | `maintenance.jpg` | Setup & Care service card |
| `services/` | `custom.jpg` | Custom Builds service card |
| `services/` | `store.png` | Full Supply Store service card (logo; use `object-contain`) |
| `partners/` | `aquaroutine.png` | Aquaroutine e-commerce partner logo (footer branding) |
| `gallery/` | `01.jpg` … `39.jpg` | Gallery grid (39 photos; 01 and 05 are large tiles) |

Until you add files, the site falls back to Unsplash placeholders.
